Voxel Engine [Dat Engine] Version 0.0.1 by Dean Prach

==Install Guide==
1. Make sure CMake is installed
2. Run the following commands:
    cmake -B build -G "MinGW Makefiles"
    cmake --build build
    cd build
    ./VoxelEngine.exe

[If cmake-tools extension is installed]
1. Press build
2. Press launch
